WebGL

컨텍스트 가져오기

WebGL을 사용하면 웹 브라우저가 설치된 기기에서 동작하는 3D 그래픽 애플리케이션을 제작할 수 있습니다. 그리고 WebGL을 사용하기 위해 별도의 플러그인 혹은 라이브러리를 설치할 필요도 없습니다. HTML Canvas와 WebGL을 지원하는 웹 브라우저라면 어디서나 실시간 3D 그래픽을 렌더링할 수 있습니다.

지금부터 WebGL을 사용하는 방법을 배워보도록 하겠습니다. 이 페이지는 여러분이 기본적인 Javascript 지식이 있다고 가정하고 있습니다. 만약 그렇지 않다면 Javascript를 먼저 배울 것을 권장합니다.

Javascript는 W3Schools 와 Mozilla Developer Network 에서 배울 수 있습니다.

HTML Canvas 준비하기 

                <body onload="main ()">
                    <canvas id="webgl-canvas">사용 중인 웹브라우저가 <canvas>를 지원하지 않습니다.</canvas>
                </body>

WebGL을 사용해 렌더링한 화면을 보여주기 위해서는 HTML Canvas가 필요하므로 위와 같이 HTML 문서에 Canvas를 추가합니다. 그리고 HTML 문서를 불러오면 WebGL을 초기화하기 위해서 <body>에 onload 이벤트를 추가합니다.

WebGL 컨텍스트 가져오기 

                function main ()
                {
                    let canvas = document.getElementById ("webgl-canvas");
                    let webgl = canvas.getContext ("webgl");
                
                    if (!webgl)
                    {
                        alert("사용 중인 브라우저가 WebGL을 지원하지 않습니다.");
                
                        return;
                    }
                
                    webgl.clearColor (0.0, 0.0, 0.0, 1.0);
                    webgl.clear (webgl.COLOR_BUFFER_BIT);
                }

가장 먼저 할 일은 HTML 문서에 있는 Canvas를 가져오는 것입니다. 그리고 Canvas로부터 WebGLRenderingContext를 가져옵니다. WebGLRenderingContext를 가져올 수 없다면 getContext 함수는 null을 반환합니다. 그리고 이러한 경우에 WebGL을 사용할 수 없음을 알려주는 창을 띄우고 종료합니다.

WebGLRenderingContext를 가져왔다면 clearColor 함수로 화면을 채울 색을 검은색으로 설정합니다. 그리고 clear 함수로 설정한 색으로 화면을 채웁니다.

축하합니다! 지금은 오직 검은 화면을 보여주는 것뿐이지만 WebGL을 사용하기 위한 준비를 모두 마쳤습니다.
